# Driver's License/Driving Permit Replacement

## Plan

### Stage 1 - Obtaining certificate from embassy
- Fill out embassy application
- Print application
- Photocopy of passport, driver's license
- Payment of 50 euros to the Embassy of Belarus
- Print receipt
- Buy 2 envelopes and Polish stamps
- Sent 15.05.24
- Reached embassy 16.05.24
- Reply received 21.07.24

### Stage 2 - Submission to urzad (office)
- Medical examination
- Photo
- Application form (wniosek)
- Sworn translation
- Fee payment at cashier
- Driver's license
- Residence permit
- Certificate of driver's license validity
- Meldunek (residence registration)
- Copy of passport
- Copy of driver's license
- Copy of residence permit
- Work contract (if meldunek is for short term)

### Stage 3 - Waiting
https://info-car.pl/new/prawo-jazdy/sprawdz-status-prawa-jazdy - check status by PESEL number

**Timeline:**
- Sent letter 15.05.24
- Reply 21.07.24
- Applied at urzad
- Received license (2 weeks after apply)

- Example: https://stach.cracow.biz/wp-content/uploads/2020/05/WZÓRPKK.pdf

**Required documents:**
- Photo
- Passport
- Residence card (karta pobytu)
- Meldunek
- Driver's license with sworn translation
- Pay state fee (can be done on-site and receipt sent by email)
- Medical examination "badanie na prawo jazdy" (to issue new driver's license for longer period than old one)
